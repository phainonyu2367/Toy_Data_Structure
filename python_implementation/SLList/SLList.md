# Single-Linked-List
One directed linked list, including nodes.

# Basic Components
- init
- addFirst: add a node at the beginning of the list
- addLast: add a node at the end of the list
- findPos: get the index of the first node with target value in the list
- remove: remove the ith node in the list
- traverse: traverse and modify the list
- get: get the value of the ith item in the list
- size: get the size of the list
- iterator: iterating pointer of the list
